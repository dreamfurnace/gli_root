# AWS GLI Helper Skill

## 스킬 설명
모든 AWS CLI 작업을 GLI 계정(917891822317)에서 안전하게 실행하는 스킬입니다.

## 주요 기능
- AWS 명령어 실행 전 자동으로 GLI 계정 전환
- 잘못된 계정에서 실행 방지
- 안전한 AWS 리소스 관리

## 사용 시나리오
- EC2 인스턴스 조회/관리
- S3 버킷 작업
- RDS 데이터베이스 관리
- 기타 모든 AWS CLI 작업

## 중요사항
❌ **절대 금지**: 직접적인 `aws` 명령어 실행
✅ **필수**: 반드시 GLI 계정 전환 후 실행
