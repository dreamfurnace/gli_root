### 🧱 UI 구성 요소 정의
## 1. 🔝 Header (상단바)
로고 / GLI 브랜드
현재 로그인된 계정(등급, 지갑주소 일부, 3자리 인증숫자)
네트워크 상태 (Web3 연결, 체인명, RPC 상태 등)
로그인 / 로그아웃 / 3일 자동 로그인 관리
언어 전환 (KR/EN)

## 2. ⬅ Left Sidebar (메인 내비게이션)
항목	설명
📄 1. Business	사업 소개 (하위 메뉴: 배경, 팀원, 전략, 일정 등 → 오토스크롤)
🔐 2. 인증 센터	회원 등급 확인 및 각 단계 인증 진행 (R/W/X)
    - GLIB(RWA코인) ↔ GLID(거버넌스) : 양방향 변환
    - GLIB(RWA코인) → GLIL(게임, 스테이블) : 단방향 변환
    - GLID(거버넌스) ↔ GLIL(게임, 스테이블) : (확인 필요)
💱 3. 토큰 변환	GLIB/GLID/GLIL 토큰 간 변환
🛍️ 4. 쇼핑몰	리조트 예약, 굳즈 구매, 식당 예약
🧑 5. 마이페이지
    - 내 회원 정보 - 등급, 이메일, 전화번호, 로그인 세션 관리 중인 장비
    - 내 레퍼럴 코드 보기, 링크 만들기
    - 내 GLI토큰(GLID, GLID, GLIL) 주소, USDT 주소 확인
    - 내 토큰&코인 별 거래내역 (리워드 입금액, 플랫폼 외부 입/출금 기록, 내부트랜젝션(토큰 사용처에서 소비) 등)
    - 블록 체인 입/출금 기능
🤝 6. 레퍼럴	내 코드 확인, 링크 공유
📑 7. 안내 센터	FAQ, 가이드 등

✅ 선택된 메뉴에 따라 중앙 Main Content가 교체됨

## 3. ➡ Right Panel (상태·도움말·상호작용, 선택적으로 출현)
단계별 인증 도움말
툴팁 가이드 제공 UX
고객센터 (Chat)

### 🎨 디자인 키워드
💠 Web3 느낌 + 고급 리조트 UX (고급스러움 + 신뢰감)
😍 이모지를 많이 사용
🟦 테마 색상: 메탈릭 블루, 골드, 다크그레이
🔒 인증은 게이미피케이션 요소와 함께 단계 표현
🪙 토큰 영역은 시각적 계량기 UI 추천 (잔액, 변환 등)

위 내용들을 기준으로 새로운 웹 화면을 개발하라. 단, ./gli_user-frontend/reff_lawide_web-frontend_reff 이 폴더에 참고해야 하는 웹 프론트 페이지를 복사해두었다. 이 폴더의 내용을 전체를 ./gli_user-frontend에 붙여넣고나서 프로젝트 이름, 데이터, api 이름 등 기존 참고 프로젝트에 대한 내용을 모두 지우고 우리 gli프로젝트의 깨끗한 프로젝트로 구성한다.
 - 작업 완료 후, reff_lawide_web-frontend_reff 폴더를 삭제하자. 
 - 대부분의 기술 규격, 라이브러리, 개발 방식, store 사용 방식 등은 참고 프로젝트(reff_lawide_web-frontend_reff)를 참고하라.
 - ./gli_api-server에 있는 django와 소통할 수 있는 웹사이트로 개발되어야 하는 것을 고려한다.
