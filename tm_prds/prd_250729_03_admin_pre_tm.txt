### gli_admin-frontend 에 웹사이트 개발

📄 PRD: GLI admin Web 프론트엔드

1. 개요 & 목적
- 비전: Web3 기반의 글로벌 리조트‑투자‑블록체인 플랫폼으로서, 프리미엄 UX와 신뢰감을 기반으로 사용자와 관리자가 seamless하게 상호작용하도록 지원
- 설명:
  · 사용자 웹(gli_user-frontend): 인증, 토큰 변환, 쇼핑몰, 마이페이지 기능 완비
  · 관리자 웹(gli_admin-frontend) : 콘텐츠 및 토큰 사용처 관리, 배분 계획 및 회원 거래 조회 기능 제공

2. 이해관계자
- 제품 책임자(PM)
- 프론트엔드 팀 (React / Next.js or Vue / Nuxt)
- 백엔드 팀 (Django API)
- 디자인 팀 (브랜드 디자인, 다크/라이트, Web3 UX)
- QA·운영팀

3. 사용자 페르소나
- Alice: Web3에 친숙, GLID/GLIB 보유, 리조트 예약 중심
- Bob: Web3 초보 사용자, 인증 가이드 필요
- Carol: 관리자, 콘텐츠 관리 및 회원 모니터링 필요

4. UX / 화면 구성

공통:
- 메탈릭 블루·골드·다크그레이 톤
- Emoji 기반 UI

기능:
  · Business 소개 콘텐츠 CRUD
  · 토큰 사용처(리조트, 굿즈 등) 관리
  · 토큰 배분 계획 설정/실행
  · 회원별 토큰 입/출금 기록 및 내부 거래 조회
  · 회원 인증 등급 상태 조회 및 승인

5. 기능 요구사항 (User Stories)
- 인증 센터:
  · 이메일/휴대폰/3D 인증 Flow 구현
  · 입금 기반 인증 (GLID, USDT, Web3 wallet)
  · 실시간 상태 반영, progress bar
- 토큰 변환:
  · 토큰 변환 인터페이스 및 계량기 UI
  · Django API 연동
- 쇼핑몰:
  · 토큰으로 예약 및 굿즈 구매
  · 예약 내역 및 토큰 차감 표시
- 마이페이지:
  · 회원정보, 레퍼럴 코드, 토큰 주소, 거래내역, 로그인 세션 관리
  · 자동 로그인(모바일 1개, 웹 1개) 유효기간 3일 로직
- 관리자:
  · 콘텐츠, 토큰 사용처 CRUD
  · 배분 계획 설정 및 실행
  · 회원 거래 및 인증 등급 조회

6. 비기능 요구사항
- Web3 통합: wagmi, ethers.js, RainbowKit 등
- REST API 통신: Axios


위 내용들을 기준으로 새로운 웹 화면을 개발하라. 단, ./gli_admin-frontend/reff_admin-frontend_reff 이 폴더에 참고해야 하는 관리자웹 프론트 페이지를 복사해두었다. 이 폴더의 내용을 전체를 ./gli_admin-frontend에 붙여넣고나서 프로젝트 이름, 데이터, api 이름 등 기존 참고 프로젝트에 대한 내용을 모두 지우고 우리 gli프로젝트의 깨끗한 프로젝트로 구성한다.
 - 작업 완료 후, reff_admin-frontend_reff폴더를 삭제하자. 
 - 대부분의 기술 규격, 라이브러리, 개발 방식, store 사용 방식 등은 GLI의 유저 웹 프로젝트(gli_user-frontend)를 참고하라.
 - ./gli_api-server에 있는 django와 소통할 수 있는 웹사이트로 개발되어야 하는 것을 고려한다.





---------(참고)-----------------------
🔹 사용자 웹: gli_user-frontend
1. Header
   · 로고 / GLI 브랜드
   · 로그인 계정 정보 (등급, 지갑주소 일부, 인증 숫자)
   · Web3 연결 상태 (체인명, RPC health)
   · 로그인/로그아웃, 자동 로그인 장비·유효기간
   · 언어 전환 버튼, 테마 전환 버튼

2. Left Sidebar (메인 내비게이션)
   1. Business – 하위 메뉴 항목 → 오토스크롤
   2. 인증 센터 – R/W/X 단계 진행 상태
   3. 토큰 변환 – GLIB ↔ GLID, GLIB → GLIL, GLID ↔ GLIL
   4. 쇼핑몰 – 리조트, 굿즈, 식당, 레저 예약
   5. 마이페이지 – 회원정보 · 레퍼럴 · 토큰 주소 · 거래내역 · 로그인 세션 관리 · 블록체인 입/출금
   6. 레퍼럴 – 내 코드 확인 및 링크 공유
   7. 안내 센터 – FAQ, 가이드

3. Right Panel (선택적)
   · 인증 단계별 도움말 및 툴팁
   · 고객센터 채팅 위젯

4. Main Content
   · 선택 메뉴에 따라 해당 컴포넌트 로드
   · 예: 인증센터 Flow, 토큰 변환 계량기 UI, 쇼핑몰 예약 화면 등
